Introducción: Aproximación numérica de los coeficientes de Fourier

Es posible calcular los coeficientes de Fourier numéricamente usando distintos métodos. Sabemos que los coeficientes de Fourier están dados por la siguiente integral:

Así pues, se puede resolver analíticamente, pero para efectos computacionales se debe realizar numéricamente, esto es, realizar una aproximación. Así pues, en esta práctica se realizó una aproximación numérica usando el algoritmo del trapecio compuesto. Este algoritmo aproxima el área bajo una curva (integral definida) de una función, con una precisión bastante considerable. Para el caso de los coeficientes de Fourier, al tener un factor que resulta ser una exponencial compleja, entonces se debe dividir en dos integrales usando la identidad de Euler y calcular el área bajo la curva de estas por separado y luego expresar como un solo número complejo. function MiTrapecio(N,a,b,f,T_0,n)
%N=número de trapecios
%a=intervalo inicial;
%b=intervalo final;
%f=funcion;
%T_0=periodo;
%n=número de armonicos;
armo=zeros(n+1,1);
cont=1;
w0=2*pi/T_0;
for i=0:n
 
            x_l=a;
            x2=a;
            s=0;
            s2=0;
            h=(b-a)/N;
            I=1;
            I2=1;
            freal=@(t) f(t)*cos(w0*i*t);
            fimag=@(t) f(t)*sin(w0*i*t);
                while I<=N-1
                    x_l=x_l+h;
                    s=s+freal(x_l);
                    I=I+1;
                end 
                while I2<=N-1
                    x2=x2+h;
                    s2=s2+fimag(x2);
                    I2=I2+1;
                end 
            Area=h/2*(freal(a)+2*s+freal(b));
            Area2=h/2*(fimag(a)+2*s2+fimag(b))*1i; 
            Area3=(Area2+Area)/T_0;
            %disp(abs(Area3));
            armo(cont)=Area3;
            cont=cont+1;
end 
Dn_Trapecio=armo;
%Programa de Lathi
T_0=pi;
N_0=256;
T=T_0/N_0;
t=(0:T:T*(N_0-1));
x_l=exp(-t/2); x_l(1)=(exp(-pi/2)+1)/2;
D_n_1=fft(x_l)/N_0;
Dn_Lathi=[D_n_1(1);D_n_1(2);D_n_1(3);D_n_1(4);D_n_1(5);];
%%Analitico 
cont=1;
Dn_analitico=zeros(n+1,1);
        for i=0:n
            x_an=0.504/(1+4*1i*i);
            Dn_analitico(cont)=x_an;
            cont=cont+1;
            
        end 
Coeficientes_Dn=['0';'1';'2';'3';'4'];
Resta_trapecio=abs(Dn_analitico-Dn_Trapecio);
Resta_Lathi=abs(Dn_analitico-Dn_Lathi);
Tabla=table(Coeficientes_Dn,Dn_analitico,Dn_Trapecio,Dn_Lathi,Resta_trapecio,Resta_Lathi)
end
